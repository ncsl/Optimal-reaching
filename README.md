# Optimal-reaching

P. Greene, A.J. Bastian, M.H. Schieber, & S.V. Sarma, Optimal reaching subject to computational and physical constraints reveals structure of the sensorimotor control system, Proc. Natl. Acad. Sci. U.S.A. 121 (14) e2319313121, https://doi.org/10.1073/pnas.2319313121 (2024).
